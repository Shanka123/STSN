# STSN : Slot Transformer Scoring Network
Official repository for the paper - "[Learning to reason over visual objects](https://openreview.net/forum?id=uR6x8Be7o_M)." 



## Requirements
* python 3.9.7
* NVIDIA GPU with CUDA 11.0+ capability
* torch==1.11.0
* torchvision==0.12.0
* glob
* PIL==8.4.0
* numpy==1.20.3
* einops==0.4.1
